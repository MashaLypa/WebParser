WebParser Version 1.0 (30-11-2014)

Создан в учебных целях.

Предназначен для поиска e-mail на сайте.
В верхнее поле вводится адрес сайта
Приложение выводит код сайта и e-mail, если таковые имеются.

Для наглядности работы при запуске приложения уже введен адрес сайта,
на котором есть много e-mail адресов.

При написании использовались С++ и Qt.
IDE Qt Creator
